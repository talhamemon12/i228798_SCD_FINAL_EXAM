# Q2 Commands

- `kubectl get pod`: Check the status of all pods in the Kubernetes cluster.
- `kubectl logs mongodb-deployment-5f6f746f74-rwmdd`: Check the logs of the MongoDB pod to verify it's starting and running correctly.
- `kubectl get pods`: Check the current status of all pods, including backend and frontend.
- `kubectl logs backend-deployment-fddf7745d-57fjr`: Check the logs of a backend pod to see why it's crashing.
- `kubectl get svc`: Check if the MongoDB service is properly defined and accessible.
- `kubectl logs frontend-deployment-7b4d94c5d8-67zqn`: Check the logs of a frontend pod to confirm it's running correctly.
